Tips Dalam Belajar Coding

Happy

Awali dengan rasa bahagia. Apapun alasan kita mempelajari coding, jika tidak menemukan kebahagiaan saat berhasil memecahkan masalah akan sulit untuk mengasosiasikan coding sebagai kegiatan menyenangkan apalagi menjadikan coding sebagai hobi.

Solve

Berpikir Kritis dalam menyelesaikan masalah, dalam beajar coding kita di tuntut untuk berpikir kritis. Oleh karena itu selalu asah kemampuan otak kita

Hobby

Coding sebagai Hobi. Jika hobi kita adalah coding dan setiap waktu luang kita gunakan melakukan coding maka keahlian coding akan berkembang pesat, dan kita akan merasa bahagia saat mengerjakannya

Action

Jika kita sudah memiliki kemauan dalam memepelajari coding maka kita harus mewujudkan keinginan kita dengan belajar tekun untuk meningkatkan kemampuan coding kita dan kita dapat mengerjakan project bersama di komunitas coding yang kita ikuti
